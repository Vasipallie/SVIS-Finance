# SVIS Finance

> "Know where your money goes."

SVIS Finance is a privacy-focused personal finance tracker built with React Native and Expo. SVIS Finance allows users to manage their wallet, track transactions, and view financial statisitics such as total income and the total expenditure in a month. Wallet data can be easily cleared or exported to other applications, making it a versatile application for users.

## Features

* Realtime Transaction updates
* Smooth User Experience
* Modern and Responsive UI
* Intuitive navigation
* LocalStorage via AsyncStorage

## üöÄ Quick Start

Follow these steps to get SVIS-Finance up and running on your local development environment.

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: `^18.x` or `^20.x` (LTS versions recommended).
- **npm**: Comes bundled with Node.js.
- **Expo Go App**: Install on your mobile device (iOS or Android) to preview the app.

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Vasipallie/SVIS-Finance.git
   cd SVIS-Finance
   ```
2. **Install dependencies**

   ```bash
   npm install
   ```
3. **Environment setup**
   Create a `.env` file in the project root by copying the example.

   ```bash
   cp .env.example .env 
   ```

   Configure your environment variables in `.env`.
